Lista de Definiciones

HTML

    Lenguaje de marcado utilizado para crear páginas web.

CCS

    Lenguaje de gojas de estilo usado para definir la apariencia de una página web.

Lista de Elementos Importantes

    • Encabezado: Usado para definir titulos y subtitulos.
    • Listas: Permiten organizar contenio en viñetas o numeraciones.
    • Enlaces: Conectan diferentes páginas o recursos.
